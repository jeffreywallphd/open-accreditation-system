# open-accreditation-system

Open source university accreditation system.

## Repository scaffold

This repository now includes an initial platform-oriented scaffold for:

- product and architecture documentation under `docs/`
- deployment and infrastructure assets under `deploy/`
- platform, security, and compliance support areas under `platform/`
- schema contracts under `schemas/`
- front-end applications under `apps/`
- back-end services under `services/`
- shared workspace packages under `packages/`
- seed, reference, and migration data under `data/`
- evidence and transfer storage areas under `storage/`
- repository-level automated test suites under `tests/`
- GitHub automation and templates under `.github/`

## Notes

- Empty directories that need to be tracked in Git include `.gitkeep` placeholders.
- Placeholder starter files were created for markdown, JSON schema, YAML, shell, TypeScript, and Docker assets so the requested structure exists in version control.
